# Hacker-News
Simple hacker news flask web by Python

- Search Hacker News의 API를 사용하여 만든 Flask Web Server
- API 사용을 최소화하기 위해 한번 Request한 페이지 내용은 JSON 파일에 저장하여 사용하는 방법으로 최적화를 고려함
